## 数据可视化总结

- 文字一行显示

```js
.box {
	/* 1.设置文本不允许换行 */
	white-space: nowrap;
	/* 2.设置盒子内容溢出隐藏 */
	overflow: hidden;
	/* 3.设置溢出隐藏文字 */
	text-overflow: ellipsis;
}
```

- 边框背景图

```js
border-image-source: url(./images/border.jpg)// 边框背景图资源
border-image-slice: 167 167 167 167; // 边框背景图裁切尺寸 省略 px
border-width: 55px 40px 25px 130px; // 边框背景图宽度
border-image-repeat: stretch 	// 图片平铺方式	 默认 stretch 拉伸平铺
// round  环绕平铺 ： 会将整个图案铺满整个边框， 图片会完整显示，但是会有很轻微的拉伸
// repeat  居中平铺。 将图片从中间开始往两头平铺。 可能导致有部分图案显示不全
 border-image: url(./images/border.jpg) 167 stretch // 连写
 border-image: url(./images/border.png) 55 40 25 130 stretch; // 不规则边框连写
// 必须先设置边框属性 且需设置内减盒子 最后需采用子绝父相把内容撑开
position: relative;
```
